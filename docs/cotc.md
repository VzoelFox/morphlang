# COTC (Core of The Core)

COTC adalah Standard Library untuk bahasa Morph.

## Modul Tersedia

### Himpunan (`cotc/himpunan`)
Implementasi Set menggunakan Hash.

- `baru()`: Membuat set kosong.
- `tambah(set, elemen)`: Menambah elemen.
- `ada(set, elemen)`: Cek keberadaan elemen.
- `hapus(set, elemen)`: Menghapus elemen (soft delete).
- `gabung_set(s1, s2)`: Union dua set.
